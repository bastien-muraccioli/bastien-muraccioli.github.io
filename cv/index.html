Publications

  • 2025.02.05
    Demonstrating a Control Framework for Physical Human-Robot Interaction Toward Industrial Applications
    Proceedings of Robotics: Science and Systems (RSS) 2025
    This paper presents a robust control framework for safe, dynamic physical human-robot interaction (pHRI) aimed at industrial applications. It integrates compliance control strategies and a second-order QP formulation to manage safety and task priorities.
  • 2025.01.21
    Scalable, Training-Free Visual Language Robotics: a modular multi-model framework for consumer-grade GPUs
    2025 IEEE/SICE International Symposium on System Integration (SII)
    The SVLR framework introduces a scalable, training-free system for vision-language-based robotic control using lightweight, open-source AI models. It interprets language commands and visual input to execute robotic actions without retraining, offering broad adaptability and low resource requirements.
  • 2022.02.13
    Can robots be good public speakers?
    Proceedings of the Sixteenth International Conference on Tangible, Embedded, and Embodied Interaction
    This paper investigates whether robots can be effective public speakers and what design considerations are critical. Using the Anki Vector robot, the authors implemented a system that translates human gestures and expressions into robot behavior, with plans to compare it to an anthropomorphic alternative.
